# Credit-Score-Prediction-using-Machine-Learning
A Machine Learning classification project that predicts customer credit scores using financial and credit-related information. The project includes data preprocessing, feature engineering, missing value handling, feature selection, model training, hyperparameter tuning with GridSearchCV, and performance comparison of multiple classification algo..
This project focuses on building a Machine Learning model to classify customer credit scores based on financial and banking information. The goal is to help financial institutions automate credit score prediction and improve decision-making. The project follows a complete ML pipeline from data preprocessing to model evaluation.

Features-->
Data Loading and Exploration
Data Cleaning and Preprocessing
Missing Value Handling
Outlier Detection and Removal
Label Encoding / Categorical Data Encoding
Feature Selection using VIF
Train-Test Split
Feature Scaling using StandardScaler
Logistic Regression
Decision Tree Classifier
Hyperparameter Tuning using GridSearchCV
Random Forest Classifier
Model Evaluation and Accuracy Comparison

Technologies Used-->
Python
Pandas
NumPy
Matplotlib
Seaborn
Scikit-learn
Jupyter Notebook

Machine Learning Workflow-->
Import Libraries
Load Dataset
Data Cleaning
Handle Missing Values
Remove Outliers
Encode Categorical Features
Feature Selection (VIF)
Train-Test Split
Feature Scaling (StandardScaler)
Train Logistic Regression, Decision Tree, and Random Forest Models
Hyperparameter Tuning with GridSearchCV
Evaluate and Compare Model Performance

Model Performance-->
Logistic Regression: 61.8% Accuracy
Decision Tree: 69.7% Accuracy
Decision Tree (GridSearchCV): 70.93% Accuracy
Random Forest Classifier: 79.7% Accuracy (Best Model)

Project Objective-->
To develop an accurate and efficient credit score classification model that predicts customer creditworthiness using financial and banking data, enabling faster and more reliable credit assessment.
